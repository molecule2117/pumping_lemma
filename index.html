The Pumping Lemma for Regular Languages

If L is regular, ∃ pumping length p such that ∀ s ∈ L with |s| ≥ p, s = xyz where:
  ① |xy| ≤ p    ② |y| ≥ 1    ③ ∀ i ≥ 0 : xyⁱz ∈ L

To prove NOT regular: Find a string where no valid xyz decomposition satisfies ③.
